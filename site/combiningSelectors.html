Combining Selectors

Element with Class Selector

These previous two span have the "Element with Class Selector" associated.

Child Selector(>)

This span element is directly inside section tag.
And this span element is inside section tag, but not directly.
This other span is out of section tag.

Descendant Selector(Space)

This paragraph element is directly inside section tag.

And this paragraph element is inside section tag, but not directly.

This other paragraph is out of section tag.

Other examples #1 (Descendant Selector - Element Class - Class Element)

This span element have class "bigRed" associated.

And this span element also has class "bigRed" associated, but he is inside p element.

Now this is a paragraph element inside of span element, which has the class "bigRed" associated with it.

And this is a paragraph element inside of span element, which does not have a class associated with it.

Other examples #2(Child Selector - Element > Class)

This span element is inside a div element, which have a class "ACG" associated.

And this span element also has class "ACG" associated, but he is inside p element.

Class Selector AND Class Selector

This span element is inside of div element, which has two classes "ACG bigRed."
This span element is inside of div element, which has one class "bigRed".
This span element has one class "ACG" and is inside of div element, which does not have any class.
